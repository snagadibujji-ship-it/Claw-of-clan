# 报告生成 Skill

将渗透测试结果整理为结构化报告，包含详细发现、PoC 脚本和修复建议。

## 报告结构

### 1. 项目概述
- 测试目标
- 测试时间
- 测试范围
- 测试方法

### 2. 执行摘要
- 高危发现概览
- 风险等级分布
- 关键建议

### 3. 详细发现
对每个漏洞：
- 漏洞名称和严重等级
- 漏洞类型
- 影响范围
- 验证步骤
- 关键证据（请求/响应/截图）
- PoC 脚本
- 修复建议

### 4. 攻击路径
- 完整攻击链图示
- 从初始访问到最终目标的路径

### 5. 附件
- PoC 脚本
- 流量抓包
- 截图证据
