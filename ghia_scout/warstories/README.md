# 🦞 War Stories — GHIA Scout 实战经验库

这里存放 GHIA Scout 的真实渗透测试/CTF 解题战报。

每份战报记录完整的攻击链：从信息收集到最终 flag，包括走了哪些弯路、关键突破口在哪里。

## 文件命名规则

```
YYYY-MM-DD_题目类型_关键词.md
```

例如：`2026-04-19_php-deserialization_regex-bypass.md`

## 战报模板

每份战报应包含：

| 区块 | 内容 |
|------|------|
| **元信息** | 日期、目标、类型、关键词、轮数、工具链 |
| **攻击链** | 每一步做了什么、发现了什么 |
| **关键突破** | 哪一步是决定性的，为什么 |
| **走的弯路** | 哪些尝试失败了，原因是什么 |
| **Payload** | 最终可复现的利用代码 |
| **经验总结** | 可迁移到同类题目的方法论 |
